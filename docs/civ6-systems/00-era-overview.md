# Civilization VI Systems — Era Breakdown

Reference for implementing Civ6-equivalent systems in Uncivilised, organized by era.

## Era Progression (Civ6)

| Era | Approx Turns (Standard) | Key Theme |
|-----|------------------------|-----------|
| **Ancient** | 1–50 | Founding, exploration, first cities, basic military |
| **Classical** | 50–100 | Expansion, early districts, first governments |
| **Medieval** | 100–150 | Religion peak, trade guilds, military diversification |
| **Renaissance** | 150–200 | Exploration, gunpowder, enlightenment |
| **Industrial** | 200–275 | Factories, railroads, nationalism |
| **Modern** | 275–350 | Ideology, world wars, mass media |
| **Atomic** | 350–425 | Nuclear, cold war, space race begins |
| **Information** | 425–500 | Victory push, GDR, future tech |

## Uncivilised Current State

The game currently runs **100 turns** with no formal era system. Most content maps roughly to **Ancient + early Classical** Civ6 eras. There are no era transitions, no era score, and no age system.

Focus on **Ancient Era first** — this is the foundation. Every system introduced here carries through the entire game. Getting Ancient right means the scaffolding for all later eras is solid.
